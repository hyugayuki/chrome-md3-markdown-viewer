# MD3 Markdown Viewer (Chrome Extension)

Chromeブラウザでローカルのマークダウンファイル（`file:///*.md`）を開いた際に、ブラウザ標準の無機質なプレーンテキスト表示を乗っ取り、Material Design 3 (MD3) 準拠の美しいマークダウンプレビュー画面へ動的に置換する Manifest V3 対応の Chrome 拡張機能です。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![MV3](https://img.shields.io/badge/manifest-v3-green.svg)

---

## 🌟 主な機能

*   **MD3 準拠のプレミアムデザイン**: 
    Material Design 3 のカラーパレット（ダークテーマ優先）を基調とした、美しく目に優しいドキュメント環境。
*   **グラスモルフィズムヘッダー**: 
    画面上部の Top App Bar には `backdrop-filter` による透過・ぼかし効果を適用し、スクロール時にコンテンツが美しく透けて見えます。
*   **階層化されたアコーディオン式目次 (TOC)**:
    本文中の見出し（H1〜H3）を自動的に解析し、親子関係を持つ木構造の目次を左サイドバーに動的生成。各項目は折りたたみ・展開が可能です。
*   **インテリジェントな ScrollSpy（自動スクロール追従）**:
    本文のスクロールに合わせて、目次の現在位置（アクティブ項目）を示す紫色の縦型スライドインジケーターが正確に滑らかに追従します。また、アクティブ項目が折りたたまれた階層の中にある場合は、自動的に親階層を展開します。
*   **目次（サイドバー）のスライド開閉**:
    目次のヘッダー、または左上のメニューボタン（三本線）をクリックすることで、サイドバーを左側にスライドして隠すことができます。サイドバーを閉じると、本文エリアがスムーズに全画面幅へ広がり、ゆったりとドキュメントを読めます。
*   **パンくずリスト (Breadcrumbs)**:
    ローカルファイルの絶対パスを解析し、上位ディレクトリへ1クリックで戻れるナビゲーションリンクを画面上部に生成します。
*   **Preview / Plain モード切り替え**:
    ヘッダーのセグメンテッドボタンにより、装飾された「Preview（ドキュメント表示）」と生のマークダウン「Plain（テキスト表示）」を簡単にスイッチできます。
*   **各種コピー機能とトースト通知 (Snackbar)**:
    生マークダウン全体のコピーや、コードブロック内の個別コードコピーがワンクリックで行えます。コピー成功時には画面下部から MD3 準拠の Snackbar が浮き上がるように出現し、完了を通知します。

## 📂 ファイル構成

```text
chrome-md3-markdown-viewer/
├── manifest.json          # 拡張機能の設定ファイル（MV3対応）
├── content.js             # MDファイルのプレーンテキスト取得・全体のUI構築・インタラクション制御
├── marked.min.js          # MarkdownをHTMLにパースする軽量ライブラリ（同梱）
├── styles.css             # MD3ベースのUIスタイリング（目次、パンくず、プレビュー）
├── sample.md              # 動作テスト用のサンプルマークダウン
├── README.md              # 本ファイル
└── icons/                 # 拡張機能アイコン画像（MD3デザイン）
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## 🚀 インストール方法（開発者モードでの導入）

Chromeウェブストアを経由せず、ローカルの開発コードから直接Chromeにインストールする手順です。

1.  本リポジトリのソースコードをローカル環境に保存します。
2.  Google Chrome を開き、URLバーに `chrome://extensions/` と入力して拡張機能管理ページを開きます。
3.  画面右上にある「**デベロッパー モード**」のスイッチを **オン** にします。
4.  画面左上に表示される「**パッケージ化されていない拡張機能を読み込む**」ボタンをクリックします。
5.  ファイル選択ダイアログが表示されるので、このプロジェクトのルートディレクトリ（`chrome-md3-markdown-viewer`）を選択して読み込みます。
6.  **[重要]** 読み込まれた拡張機能カード内の「**詳細**」ボタンをクリックし、設定ページ下部にある「**ファイルの URL へのアクセスを許可する**」のトグルスイッチを **オン** に変更します。
    *   *※この設定を行わないと、ローカルの `file://` スキーム上のマークダウンファイルに対して拡張機能が動作しません。*

## 📝 使い方

1.  Chromeブラウザで、ローカルにある任意の `.md`（または `.markdown`）ファイルを開きます（ファイルをブラウザへドラッグ＆ドロップでも起動します）。
2.  自動的に MD3 の美しいプレビュー画面へと切り替わります。
3.  表示を確認する際は、同梱の `sample.md` を Chrome にドラッグ＆ドロップして動作をお試しください。
